
We need version 2.1.2 of Keras

To uninstall your version of Keras and reinstall Keras v2.1.2 :   
$pip uninstall keras   
$pip install keras==2.1.2   


To run the program :   
SVM model : 
$python svm.py  
or   
$python3 svm.py   

For the other deep learning models   
$python main.py   
$python3 main.py
